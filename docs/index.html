Welcome to ToDo App

Organize your tasks easily! Here’s how:

🔹 Priority Levels:

  • 1 → High Priority
  • 2 → Medium Priority
  • 3 → Low Priority (default if no number is given)

📝 How to Add: Type your task starting with the priority number. Example: 1 Finish project report.

✅ Mark as Done: Check the box next to a task.

🗑️ Delete: Click the trash icon to remove a task.
